# Jad Elsaghir

Personal portfolio and consulting website for Jad Elsaghir, Industrial Engineer and founder of the developing Crux Consulting practice.

## Local development

The site is intentionally dependency-free. Serve the repository root with any static server:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Deployment

GitHub Actions deploys the static site to [jadelsaghir.github.io](https://jadelsaghir.github.io/) on pushes to `main`.
